# Design Steps for FreeCAD Benchy

1. **Setup**: Initialize FreeCAD and create a new document.
2. **Hull Creation**:
   - Create the base shape of the hull.
   - Apply the 40° bow overhang.
   - Shell or subtract to form the interior.
3. **Deck**:
   - Create the main deck surface.
   - Add the cargo box (12x10.81mm outside, 8x7mm inside, 9mm deep).
4. **Cabin (Bridge)**:
   - Construct the cabin walls.
   - Add the roof with 5.5° inclination.
   - Cut out windows (10.5x9.5mm front, 9mm diameter rear).
5. **Chimney**:
   - Create the cylindrical chimney (7mm outer, 3mm inner, 11mm deep).
6. **Details**:
   - Add hawsepipe (4mm inner diameter).
   - Add nameplate on stern.
7. **Finalization**: Export or save the model.
