# Benchy Design Plan

## 1. Hull
- Create a sketch for the hull profile.
- Use a loft or sweep to create the curved hull.
- Bow overhang should be 40°.

## 2. Deck
- Add a flat deck on top of the hull.

## 3. Cabin
- Create a box for the cabin.
- Add windows.
- Add a roof with a 5.5° inclination.

## 4. Chimney
- Create a cylinder on top of the cabin.
- Add a hole in the center.

## 5. Verification
- Measure the bounding box: 60x31x48 mm.
- Export to STL.
- Generate TechDraw PDF with Top, Front, and Side views.
